InputNumber数字输入框

InputNumber数字输入框
使用方法
全局配置                
layui.config({
    base:'../js/lay-module/',
    version:true
}).extend({
    fcInputNumber:'fcInputNumber/fcInputNumber',
});

页面中调用
layui.use(['fcInputNumber'],function(){
    var fcInputNumber = layui.fcInputNumber;
});
元素结构
<input class="fc-input-number"  />
Demo
基本输入框
参数：value="0"　　step="1"　　max="5"　　min="-5"
改变参数：step="10",max="100",min="0"
<input class="fc-input-number" fc-filter="test2" />
在需要渲染参数的input上，添加标识 fc-filter，render中可以放多个数组对象 
								fcInputNumber.render([
									{"test2": {"step" : 10, "min" : 10, "max" : 100, "value":10}},
									{"test3": {"step" : 10, "min" : 10, "max" : 100, "value":10}},
								]);
自定义步数
参数：value="0"　　step="5"　　max="20"
自定义步数的需要添加 readonly属性设置只读
精度
参数：value="0.1"　　step="0.1"　　max="1.0"　　precision="1"
禁止
参数：value="10"　　disabled
改变状态后，进行渲染：fcInputNumber.render();
Size
三种大小的数字输入框，当 size 分别为 large 和 small 时，输入框高度为 40px 和 24px ，默认高度为 32px。
控制按钮位置
控制按钮位置:默认左右， position="right" 右侧
